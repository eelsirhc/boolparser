Bool Parser

The boolparser library defines a simple boolean parser that supports logical operators and parentheses (and not much else) thanks to the PyParsing library. The library allows you write a simple parser that can parse string from “a>0″ to “(a==b)&(c>1)|(d==0)&((e<=1)&!f)” once you’ve defined what to do with the named variables.

The manipulation of user variables is defined by the `EvaluateVariable` class through an `eval` function interface. Overriding this class makes it possible to customize the behavior of the parser (e.g. to look/store the variable in a dictionary or file).

Example

To setup the parser you need to define an object inherited from EvaluateVariable that provides (at least) the eval function to convert an undefined variable name into data, and sometimes define an object that describes the behavior of logical comparisons (such as <=, <, ==, >, >=, !) for that variable.

For example, the parser can filter a NetCDF file based on variables in the file without ‘hard-coding’ the variables in the code. In this case, we need a class that stores NetCDF file and, when given a variable name, returns the variable data. The comparisons are typical value comparisons that the NumPy library already handles so we shouldn’t need the ‘comparator object’ 


    import netCDF4
    from boolparser.core import BoolParser, EvaluateVariable
    import sys

    def define_class(nc):
        class ev_nc(EvaluateVariable):
            def eval(self):
                if self.value in nc.variables:
                    return nc.variables[self.value][:]
                elif self.value in nc.dimensions:
                    return nc.dimensions[self.value][:]
                else:
                    raise NameError("Not a variable: {0}".format(self.value))
                    return self.value
        return ev_nc

    nc = netCDF4.Dataset("mcs.year1.nc")
    nceval = define_class(nc)
    bp   = BoolParser(nceval)

    line = """(Profile_lat>-6)&(Profile_lat<-5)&(Profile_lon>137)&(Profile_lon<138)"""
    #equivalent to np.where((nc.variables["Profile_lat"][:]>-6)&                                                                                                                                         
    #                   (nc.variables["Profile_lat"][:]>-5)&                                                                                                                                         
    #                   (nc.variables["Profile_lon"][:]>137)&                                                                                                                                        
    #                   (nc.variables["Profile_lon"][:]<138))                                                                                                                                        

    res = bp.parse(line)
    if res.any():
        print nc.variables["T_surf"][res]
    else:
        print "no data"
